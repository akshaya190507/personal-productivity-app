# 🚀 Personal Productivity App

A full-stack productivity tracking web application built using Node.js, Express, MongoDB Atlas, and JWT authentication.

---

## 🌐 Live Demo

👉 https://personal-productivity-app.onrender.com

---

## 🛠 Tech Stack

- Node.js  
- Express.js  
- MongoDB Atlas  
- Mongoose  
- JWT Authentication  
- Render (Deployment)  
- Git & GitHub  

---

## ✨ Features

- 🔐 User Authentication (JWT-based)
- ➕ Create tasks
- ✏️ Update task status (Pending / In Progress / Done)
- ❌ Delete tasks
- 📊 Task statistics dashboard
- 📅 Weekly productivity tracking
- 🛡 Protected API routes
